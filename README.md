# Smoothie96
This is a port of my theme **Smoothie** for [Windows96](https://windows96.net) (v3 only).

**This theme is still incomplete - there may still be oversights I haven't corrected.**

## How do I install it?
__**[Set up my personal package repository](https://github.com/Driftini/w96pkgs/blob/main/README.md)**__, then install *Smoothie96 (UNSTABLE)* from the Package Manager.

Reboot Windows96 and select the theme in the *Display Properties* (right-click the desktop > *Properties*).
